Weather API

This API provides weather data.

Specify location and date and the API will return humidity, temperature, wind speed, and wind direction.

The API works through GET requests to:

http://myapi.com/

It will access a weather database (sample data at this stage), and return a JSON response with the data requested. It will also include the units used to record each parameter.

Example Calls

All Parameters

To see all four parameters, simply specify location and date.

For example, to see weather for Cardiff on 23rd April:

http://myapi.com/get-weather?location=Cardiff&date=2023-04-23

Response

[ { "date": "2023-04-23", "humidity": 60.0, "location": "Cardiff", "temperature": 20.0, "wind_direction": null, "wind_speed": 10.5 }, { "units": { "humidity": "percentage", "temperature": "celsius", "wind_direction": "degrees", "wind_speed": "miles per hour" } } ]

Specified Parameters

Parameters can be specified, they just need to be included in the request.

They are:

  • temperature
  • humidity
  • wind_speed
  • wind_direction

For example, to see only temperature and wind speed for Cardiff on 23rd April:

http://myapi.com/get-weather?location=Cardiff&date=2023-04-23&parameters=humidity,wind_speed

Response

[ { "date": "2023-04-23", "humidity": 60.0, "location": "Cardiff", "wind_speed": 10.5 }, { "units": { "humidity": "percentage", "wind_speed": "miles per hour" } } ]
